# Battle Simulator

## Project Description
Battle simulator is a turn base fighting game. Create your character, and fight various enemy to level up!

## Execution and usage
Run the code using run code button or run
```sh
python main.py
```

## Used Techonologies
---
+ pandas
```sh
pip install pandas
```
+ numpy
```sh
pip install numpy
```
+ matplotlib
```sh
pip install matplotlib
```

## Current Features 
+ Character stats analytic
+ turn base battle
+ Creating character
